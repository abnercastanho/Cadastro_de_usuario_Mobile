# 📱 Cadastro de Usuário Mobile

Projeto desenvolvido durante a aula de **Desenvolvimento Mobile**, com o objetivo de criar uma interface simples para cadastro de usuários em Android.

---

## 🧠 Descrição do Projeto

O aplicativo permite:

* Inserir **nome completo**
* Inserir **email**
* Salvar os dados informados
* Exibir os usuários cadastrados em uma lista

---

## 🛠️ Tecnologias Utilizadas

* Java
* Android Studio
* XML (Layouts)
* Material Design Components

---

## 📂 Estrutura do Projeto

### 🔹 Tela de Cadastro (`creat_user.xml`)

* Campo para nome completo
* Campo para email
* Botão "Salvar"

### 🔹 Tela Principal (`MainActivity`)

* Lista de usuários (RecyclerView)
* Botão "Cadastrar"

### 🔹 Classe `UserAdapter`

Responsável por:

* Capturar os dados inseridos
* Validar os campos
* Enviar os dados para a lista principal

---

## 📋 Funcionalidades

* Cadastro de usuário
* Validação de campos
* Armazenamento temporário em lista
* Navegação entre telas
